# TaskFlow

A simple Task Management application built with **React** (frontend) and **Node.js** (backend).

##  Features

* Create new tasks 
* View all tasks 
* Update task details 
* Delete tasks 

##  Tech Stack

* **Frontend**: React With Typescript
* **Backend**: Node.js, Express
* **Data Storage**: in-memory data structure to store tasks [for now]

##  Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
